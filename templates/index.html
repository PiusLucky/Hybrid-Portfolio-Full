Road map

Most of the conventions I follow upon any new projects are as follows:
  • Drafting a proper UML for high-powered web applications.
  • Following Google's style guide
  • Leveraging responsive web frameworks to consistently complete product deliverables ahead of schedule.
  • Using jQuery and leveraging popular plugins for common or complex interactive components.
  • Engineering the Backend, adhering to Modularization
  • Engineering the Backend, adhering to Unit Testing
  • Building RESTful APIs (if neccessary)
  • Refactorization via Code cleaning
